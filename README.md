# 🚀 AI Text Summarizer

A modern AI-powered text summarization web application built using HTML, CSS, JavaScript, and Groq API with Llama 3 integration.

Users can paste large text content and instantly generate concise AI summaries with a clean, responsive, glassmorphism-inspired interface.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/ai-text-summarizer.git
```

---

## 2️⃣ Open Project Folder

```bash
cd ai-text-summarizer
```

---

## 3️⃣ Add Your Groq API Key

Open:

```bash
script.js
```

Replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

with:

```javascript
const API_KEY = "gsk_xxxxxxxxx";
```

---

## 4️⃣ Run Project

Open:

```bash
index.html
```

OR use VS Code Live Server extension.

# 🧠 How It Works

1. User enters text into textarea
2. JavaScript captures the input
3. Fetch API sends request to Groq API
4. AI model generates summary
5. Summary is displayed instantly on screen
